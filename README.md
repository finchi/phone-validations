# Phone::Validations

Validates Austrian phone numbers

## Installation

Add this line to your application's Gemfile:

    gem 'phone-validations'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install phone-validations

## Usage

    "004369912333212".is_austrian_mobile_number?
